Undelete Flow

StepActionCode
1Confirm dialogTitle: 'กู้คืนร้าน "{shopName}"?'

Uses shops.owner_id to correctly identify the owner (not just any user with shop_id)

Days Overdue Calculation (code)

getDaysOverdue(shop):
- if (!shop.subscription_paid_until) return 0
- paidUntil = new Date(subscription_paid_until), today = new Date()
- both setHours(0,0,0,0)
- return Math.max(0, Math.floor((today - paidUntil) / 1day))
